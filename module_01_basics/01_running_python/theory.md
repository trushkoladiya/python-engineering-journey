# Running Python

## What is Python?

Python is a programming language. You write instructions in a file, and the computer follows them.

## 3 Ways to Run Python

### 1. Python File (Script)

- Create a file with `.py` extension (e.g., `hello.py`)
- Write your code inside it
- Run it from terminal:

```
python3 hello.py
```

### 2. Interactive Mode (REPL)

- Open terminal and type `python3`
- You get a `>>>` prompt
- Type code line by line and see results instantly

```
>>> 2 + 3
5
>>> print("hello")
hello
```

- Type `exit()` to leave

### 3. Script Execution

- You write all your code in a `.py` file
- Run the whole file at once
- This is the most common way to write real programs

## Which Should You Use?

| Method | Best For |
|--------|----------|
| REPL | Quick testing, trying small things |
| .py file | Writing real programs |

## Key Points

- Python files end with `.py`
- Use `python3 filename.py` to run a script
- REPL is great for experimenting
- Every Python programmer uses both methods
