Hey! I'm **Trush Koladiya**, a student.

I decided to learn Python properly — not random tutorials, not jumping around — but in a **structured, step-by-step way**. So I built this entire repo while learning.

Every topic has:

## 🚀 How to Use

### 1. Fork & Clone

```bash
git clone https://github.com/trushkoladiya/python-full-course.git
cd python-full-course
```

### 2. Start from Module 1

```bash
cd module_01_basics/01_running_python
```

### 3. For each topic:

1. Read `theory.md` — understand the concept
2. Run `code.py` — see it working
3. Try the challenges at the bottom of each `code.py`
4. Move to the next topic

```bash
# Read theory
cat theory.md

# Run code
python3 code.py
```

> ⚠️ **Follow the order!** Module 1 → 2 → 3 → ... → 18. Each topic builds on the previous one.
