# Contributing

Hey! Thanks for checking this out.

I'm a student, not a professional — so if you find any mistakes or want to improve something, feel free to:

1. **Fork** this repo
2. **Fix** the issue
3. **Submit a Pull Request**

### What you can help with:
- Fix typos or errors in theory/code
- Improve explanations to make them clearer
- Add more practice challenges
- Report bugs (open an Issue)

### Rules:
- Every `code.py` must be runnable (`python3 code.py` should work)
- Keep explanations simple and beginner-friendly
- Don't use concepts from future modules (each topic builds on previous ones)

Thanks! 🙏
